✈️ ERP 機位擷取器

一鍵擷取全部機位資料，自動產出狀態報告

📌 第一步：安裝書籤（僅需一次）

✓ 已複製到剪貼簿，繼續下方步驟
📊 ERP 機位報告（拖曳備用）
複製後，依下列步驟手動新增書籤：
1
按 Ctrl+Shift+B 顯示書籤列（已顯示可跳過）
2
在書籤列空白處按右鍵 → 選「新增書籤...」
3
名稱填：ERP 機位報告
4
網址欄按 Ctrl+A 全選後，Ctrl+V 貼上
5
點「儲存」完成

🚀 每次使用方式

  1. 用 Chrome 登入 ERP，開啟 SearchList 頁面（停在第 1 頁）
  2. 點擊書籤列的「ERP 機位報告」
  3. 畫面出現深色進度浮層，等待約 20～40 秒（自動翻頁擷取中）
  4. 完成後自動開新分頁顯示報告，或下載 .html 報告檔
